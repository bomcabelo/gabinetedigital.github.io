Instalação¶

Para instalar o Gabinete Digital, vamos necessitar de vários pré-requisitos

Detalhamento¶

O que você necessita ter já pronto

  • Servidor Ubuntu Server Linux (versão testada nesta documentação 14.04) com permissão de root

Nota

Seu servidor deve estar conectado a internet

O que vamos instalar e configurar

  • Nginx webserver
  • Python
  • PHP
  • Ruby
  • MySQL

Softwares e frameworks de terceiros

  • Wordpress
  • Allourideas e Pairwise
  • Thumbor
  • Nextgen Gallery (plugin WP)
